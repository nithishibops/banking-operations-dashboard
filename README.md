# 🏦 Banking Operations Dashboard

## 📌 Project Overview

The **Banking Operations Dashboard** is an interactive Power BI solution designed to analyze banking performance through customer, deposit, loan, and branch data. The dashboard provides a comprehensive view of key banking metrics, helping stakeholders monitor operational performance and make informed business decisions.

---

## 🎯 Project Objective

The primary objective of this project is to:

* Monitor customer growth and banking operations.
* Analyze deposits and loan performance.
* Evaluate branch-wise business performance.
* Track financial KPIs and operational trends.
* Support strategic decision-making using business intelligence.

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Power Query
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset includes banking-related information such as:

* Customer ID
* Customer Name
* Branch
* Account Type
* Join Date
* Deposit Amount
* Loan Amount
* Account Status
* City
* Transaction Details

---

## 📈 Key Performance Indicators (KPIs)

* Total Customers
* Total Deposits
* Total Loans
* Average Deposit Amount
* Average Loan Amount
* Customer Growth
* Branch Performance

---

## 📊 Dashboard Features

* Executive KPI Cards
* Customer Growth Analysis
* Deposit Analysis
* Loan Analysis
* Branch-wise Performance
* Account Type Distribution
* Monthly Banking Trends
* Interactive Filters and Slicers

## 💡 Business Insights
